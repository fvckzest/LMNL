# User Journey Walkthrough: LMNL Digital Ticketing

Below maps attendee touchpoints from digital acquisition pathways up to physical checkpoint scans.

---

## Step 1: The Access Request
**A. Direct Purchase (Public):**
- Customer selects admission directly on the Space portal.
- Redirects smoothly into authenticated Square payment gates.

**B. Invite Approvals (Private):**
1. Guest requests via the dashboard form.
2. Admins approve targets via dashboard queries.
3. Automation routes custom checkout instructions directly into guest inboxes.

---

## Step 2: The Webhook Handshake
*Happens instantaneously in server-side processing contexts.*
- Square verifies transaction logs securely.
- Background workers capture IDs routing details onto persistent infrastructure.

---

## Step 3: The Dispatch Protocol
A branded dispatch lands securely addressing clients directly:
- **Apple Clients:** Click to add to Apple Wallet natively.
- **General Access:** Provides a unique URL link.

---

## Step 4: Event Check-In
Guests arrive displaying either the `.pkpass` file or standard QR codes on browser panels. Staff verification sweeps map data to entry records securely.
